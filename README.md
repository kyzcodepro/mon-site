# 🍗 Farmer Crousty

Site vitrine animé pour **Farmer Crousty**, l'enseigne du poulet croustillant.
Page d'accueil immersive, gourmande et pleine d'animations conçue pour donner faim.

## ✨ Au menu des animations

- **Preloader** « rideau » qui se lève au chargement
- **Hero animé** : dégradé feu en mouvement, particules (canvas), emojis flottants en parallax à la souris
- **Bandeau défilant** (marquee) infini
- **Compteurs animés** au scroll
- **Reveals** progressifs (IntersectionObserver)
- **Cartes menu** avec effets de survol (tilt, zoom, glow)
- **Showcase « sizzle »** type vidéo : poulet qui grésille + vapeur animée
- Header sticky, menu mobile, bouton retour en haut
- 100 % autonome (aucune dépendance build), compatible `prefers-reduced-motion`
